**1\. 公钥密码算法设计中需要寻找什么样的函数进行加解密？**  
公钥密码算法需基于**陷门单向函数（Trapdoor One-way Function）**：

*   给定明文和公钥，**加密容易**；
*   给定密文和公钥，**解密在计算上不可行**；
*   但若**知道陷门信息（即私钥）**，则解密**容易**。  
    该函数在不知道陷门时是单向的，知道陷门后可高效求逆。

* * *

**2\. 背包密码体制的思想是什么？**  
背包密码体制基于**背包问题的计算困难性**：

*   **加密**：将明文看作0/1向量，与公开的背包向量做内积得到密文，**计算容易**；
*   **解密**：攻击者需从密文和公钥背包向量反推明文，等价于求解**一般背包问题**，**计算困难**；
*   **合法接收者**通过**陷门信息**（超递增序列、模数μ和ω⁻¹）将密文还原为**简单背包问题**，**可用贪心算法高效求解**。  
    即：**将易解的超递增背包伪装成难解的一般背包作为公钥**。

**3\. 简述RSA密码体制中公钥、私钥的产生过程，加解密的方法。举例说明。**

**密钥生成过程**：

1.  随机选取两个**大素数** ( p ) 和 ( q )；
2.  计算 ($n = p \times q$ )，( $\psi(n) = (p-1)(q-1)$)；
3.  选整数 ( e )，满足 ($1 < e < \psi(n)$) 且 ( $\gcd(e, \psi(n)) = 1$)；
4.  计算 ( d )，满足 ( $d \cdot e \equiv 1 \pmod{\psi(n)}$)；
5.  **公钥为 ( (e, n) )**，**私钥为 ( (d, n) )**（( p, q ) 保密）。

**加解密方法**：

*   **加密**：密文 ($ c = m^e \bmod n $)；
*   **解密**：明文 ($ m = c^d \bmod n $)；
*   要求明文 ( m < n )。

**PPT举例**：

*   选$ ( p = 43 )，( q = 59 ) → ( n = 2539 )，( \psi(n) = 2436 )；$
*   选 ( e = 13 )，解得 ( d = 937 )；
*   明文 "pu" → 数字化为 1520；
*   $加密：( c = 1520^{13} \bmod 2539 = 95 )；$
*   $解密：( m = 95^{937} \bmod 2539 = 1520 ) → 还原为 "pu"。$
